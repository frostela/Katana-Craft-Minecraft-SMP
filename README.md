# Katana-Craft-Minecraft-SMP
Personal Minecraft SMP on Paper 26.1.2 with plugins, datapacks, and a private Discord chat sync bot.

**IP (Java):** `kc.frostela.com`  
**Bedrock Port:** `19132`  
**Version:** 26.1.2 (Paper + Plugins + Datapacks)

Katana Craft is a Minecraft SMP hosted on **Oracle Free Tier VM**, running on a
Pterodactyl game panel and managed over SSH using PuTTY.  
This project is built with passion and curiosity — learning server hosting,
Linux, panel deployment, and automation along the way.

### 🧩 Plugins Used (Paper / Spigot)
- EconomyShopGUI, FancyHolograms, FancyNPCs
- AdvancedTeleport, BetterRTP, Chunky, CommandAPI
- CoreProtect, CustomMotd, DiscordSRV, FAWE (FastAsyncWorldEdit)
- Geyser-Spigot (Java + Bedrock Support)
- GriefPrevention, GWarp, ImageFrame, InvisibleFrames
- InvSeePlusPlus, LuckPerms, nLogin, Oraxen
- PhoenixCrates, PlaceholderAPI, PlayerKits2
- ProtocolLib, QuickShop-Hikari, ResourcePackManager
- Sit, SkinsRestorer, Spartan, TAB
- UltimateTeams, Vanish, Vault, ViaVersion, VoiceChat
- Votifier, VotingPlugin, WorldGuard, XConomy

### 🔗 Discord Bot (Private)
This repo also contains a private Discord-Minecraft chat bridge bot.
Its function:  
➡ Sync messages between a specific Discord channel and the Minecraft server chat.
